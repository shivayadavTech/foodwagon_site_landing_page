# 🍔 FoodWagon - Figma to Responsive HTML Conversion

A professional food delivery landing page meticulously converted from a Figma design into high-quality code. This project focuses on pixel-perfect accuracy and a seamless responsive layout.

## 🚀 Live Demo
You can view the live project here:https://shivayadavtech.github.io/foodwagon_site_landing_page/

## 🛠️ Tech Stack Used
* **HTML5** - For semantic web structure.
* **CSS3** - For custom styling, layout, and smooth animations.
* **Bootstrap 5** - For a robust responsive grid and UI components.
* **Besic JavaScript** - Powering interactive elements like popup modals and navigation.

## ✨ Key Features
* **Figma to Code:** Exact pixel-perfect conversion from the original design.
* **Fully Responsive:** Optimized performance across Mobile, Tablet, and Desktop screens.
* **Clean & Validated Code:** W3C validated, well-commented, and organized CSS/HTML structure.
* **Interactive UI:** Features smooth scrolling effects and dynamic functional popups.
